### 初始化

- 安装依赖
```shell
donet restore
```

- 初始化子模块
```shell
git submodule update --init --recursive
```


### 命令速查
- 移除子模块
```shell
git submodule deinit XXXXXX
git rm XXXXXX
rm -rf .git/modules/XXXXXX
```
